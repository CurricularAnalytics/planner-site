# Admin Wireframe

This branch contains the admin dashboard wireframe for the Curricular Analytics degree planner project.

## Purpose

The admin view is designed for academic leadership to monitor:
- retention and completion metrics
- degree planner progress by college
- advising coverage and capacity
- critical course bottlenecks
- executive reports and analytics

## Current Admin Features

- sticky filter summary and filter controls
- priority attention section
- institutional retention and completion metric cards
- past-year comparison metrics
- updated admin terminology:
  - Transfer 2-Year Graduation Rate
  - Two-Year FTFT Rate
  - Total Courses Offered
  - Active Students' Degree Completion Plans
- Degree Planner Progress by College chart
- cohort-aligned colors:
  - light blue = On Track
  - dark blue = Need Support
  - red = Off Track
- advising coverage and capacity panel
- critical courses section
- executive reports and export actions

## Tech Stack

- Vue 3
- Vite
- JavaScript
- Plain CSS
